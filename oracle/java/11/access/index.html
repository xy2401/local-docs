如何使用Java Access Bridge，Java Accessibility API（JAAPI）和Java Accessibility Utilities创建可访问的应用程序。Java Access Bridge使某些Java应用程序对Microsoft Windows上的辅助技术可见。JAAPI使您能够创建残障人士可以访问的Java应用程序。Java Accessibility Utilites是一组实用程序类，可帮助辅助技术提供对实现JAAPI的GUI工具包的访问。
