Java管理扩展（JMX）：跟踪的结尾

您已经到达“ Java管理扩展（JMX）”跟踪的结尾。

如果您对此步道有意见或建议，请使用我们的反馈页面来告诉我们。

本教程的目的是向您简要介绍JMX技术的基本元素。在其他地方可以找到更高级的材料。

如果您想进一步研究JMX技术，以下资源将非常有用：

  • Java SE平台的JMX技术文档 。JSE技术文档包含在Java SE技术文档中 。
  • JMX技术概述 。用于JMX技术的Java SE平台文档包含“ JMX技术概述”，该概述提供了对JMX技术的详细描述，而不是在本教程中提供的。
  • JMX技术教程 。本教程中的课程基于“ JMX技术教程”中使用的示例。除了本教程中介绍的基本元素之外，“ JMX技术教程”还介绍了JMX技术的更多高级方面。
  • 除了以上教程中介绍的示例之外，一旦安装了Java Development Kit（JDK）6，就可以在以下目录中找到一个示例应用程序，该示例应用程序演示了JMX API的实际实现：
    jdk_home/sample/jmx/jmx-scandir
    哪里jdk_home是JDK软件的安装目录。的jmx-scandir example是一个高级示例，它提供了实际场景中JMX API的高级概念。
  • Java平台的监视和管理 。用于监视和管理Java SE平台的技术和API严重依赖于JMX技术。《 Java SE监视和管理指南》包含有关使用JMX技术进行监视和管理的章节。
  • Java管理扩展（JMX）技术 。JMX技术主页是有关JMX技术的所有最新新闻的地方。
