使用预定义格式

版本说明：此日期和时间部分使用java.util包中的日期和时间API。JDK 8版本中提供的java.time API提供了全面的日期和时间模型，该模型对java.util类进行了重大改进。在Date Time跟踪中描述了java.time API。“ 传统日期时间代码”页面可能会特别令人感兴趣。

的DateFormat class允许您以对语言环境敏感的方式使用预定义的样式来格式化日期和时间。以下各节说明了如何使用DateFormat一个名为“程序”的类DateFormatDemo.java 。

日期

用格式化日期DateFormat上课是一个两步过程。首先，您可以使用getDateInstance方法。其次，您调用format方法，该方法返回一个String包含格式化日期。下面的示例通过调用这两个方法来格式化今天的日期：

Date today;
String dateOut;
DateFormat dateFormatter;

dateFormatter = DateFormat.getDateInstance(DateFormat.DEFAULT, currentLocale);
today = new Date();
dateOut = dateFormatter.format(today);

System.out.println(dateOut + " " + currentLocale.toString());

此代码生成的输出如下。请注意，日期格式随Locale 。以来DateFormat对语言环境敏感，它会照顾每个格式的详细信息Locale 。

30 juin 2009     fr_FR
30.06.2009       de_DE
Jun 30, 2009     en_US

前面的代码示例指定了DEFAULT格式样式。的DEFAULT样式只是预定义的格式样式之一， DateFormat该类提供如下：

  • 默认
  • 短
  • 介质
  • 长
  • 充分

时报

Date对象代表日期和时间。用格式化时间DateFormat该类与设置日期格式相似，不同之处在于，您使用getTimeInstance方法如下：

DateFormat timeFormatter =
    DateFormat.getTimeInstance(DateFormat.DEFAULT, currentLocale);

日期和时间

以相同的日期和时间显示String ，使用getDateTimeInstance方法。第一个参数是日期样式，第二个参数是时间样式。第三个参数是Locale 。这是一个简单的示例：

DateFormat formatter = DateFormat.getDateTimeInstance(
                           DateFormat.LONG, 
                           DateFormat.LONG, 
                           currentLocale);
