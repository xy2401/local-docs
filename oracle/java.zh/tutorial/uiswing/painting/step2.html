创建演示应用程序（步骤2）

接下来，我们将自定义绘图表面添加到框架。为此，我们将创建一个子类javax.swing.JPanel （一个通用的轻量级容器），它将提供用于渲染自定义绘画的代码。

一个javax.swing。JPanel子类

单击启动按钮，以使用Java™Web Start （ 下载JDK 7或更高版本 ）运行SwingPaintDemo2。或者，要自己编译并运行示例，请参考示例索引 。


package painting;

import javax.swing.SwingUtilities;
import javax.swing.JFrame;
import javax.swing.JPanel;
import javax.swing.BorderFactory;
import java.awt.Color;
import java.awt.Dimension;
import java.awt.Graphics;

public class SwingPaintDemo2 {
   
    public static void main(String[] args) {
        SwingUtilities.invokeLater(new Runnable() {
            public void run() {
                createAndShowGUI(); 
            }
        });
    }

    private static void createAndShowGUI() {
        System.out.println("Created GUI on EDT? "+
        SwingUtilities.isEventDispatchThread());
        JFrame f = new JFrame("Swing Paint Demo");
        f.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        f.add(new MyPanel());
        f.pack();
        f.setVisible(true);
    }
}

class MyPanel extends JPanel {

    public MyPanel() {
        setBorder(BorderFactory.createLineBorder(Color.black));
    }

    public Dimension getPreferredSize() {
        return new Dimension(250,200);
    }

    public void paintComponent(Graphics g) {
        super.paintComponent(g);       

        // Draw Text
        g.drawString("This is my custom Panel!",10,20);
    }  
}

您会注意到的第一个变化是，我们现在正在导入许多其他类，例如JPanel ， Color和Graphics 。由于某些较旧的AWT类仍在现代Swing应用程序中使用，因此看到java.awt包装在一些导入语句中。我们还定义了一个自定义JPanel子类，称为MyPanel ，其中包含大部分新代码。

的MyPanel类定义具有一个构造函数，该构造函数在其边缘设置了黑色边框。这是一个微妙的细节，一开始可能很难看清（如果是的话，只需注释掉对setBorder然后重新编译。） MyPanel也覆盖getPreferredSize ，该值将返回面板的所需宽度和高度（在这种情况下，宽度为250，高度为200。）因此， SwingPaintDemo类不再需要指定帧的大小（以像素为单位）。它只是将面板添加到框架中，然后调用pack 。

的paintComponent方法是您进行所有自定义绘画的地方。此方法定义为javax.swing.JComponent然后由您的子类覆盖以提供其自定义行为。它的唯一参数是java.awt.Graphics对象，提供了许多绘制2D形状和获取有关应用程序图形环境的信息的方法。在大多数情况下，此方法实际接收的对象将是java.awt.Graphics2D （一种Graphics子类），为复杂的2D图形渲染提供支持。

大多数标准的Swing组件的外观都由单独的“ UI Delegate”对象实现。的调用super.paintComponent(g)将图形上下文传递给组件的UI委托，该委托绘制面板的背景。有关此过程的详细信息，请参见上述SDN文章中标题为“绘画和UI委托”的部分。

练习：

  1. 现在您已经在屏幕上绘制了一些自定义文本，请像以前一样尝试最小化和还原应用程序。
  2. 用另一个窗口遮挡一部分文本，然后将该窗口移开以重新显示自定义文本。在这两种情况下，喷漆子系统都将确定该组件已损坏，并确保您的paintComponent方法被调用。
