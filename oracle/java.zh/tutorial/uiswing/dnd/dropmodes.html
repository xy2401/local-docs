设置放置模式

在列表等组件上启用放置时，需要确定如何解释放置位置。例如，是否要限制用户替换现有条目？您是否只允许添加或插入新条目？您想同时允许两者吗？要配置此行为， JList类提供了setDropMode支持以下放置模式的方法。

  • 的默认放置模式JList是DropMode.USE_SELECTION 。在此模式下拖动时，列表中的选定项目将移动以呼应潜在的下降点。在放置上，所选项目将移动到放置位置。提供此模式是为了向后兼容，否则不建议使用。
  • 在DropMode.ON ，列表中的所选项目将移动以呼应潜在的放置点，但是所选项目不会在放置位置受到影响。此模式可用于放置在现有列表项的顶部。
  • 在DropMode.INSERT ，则用户只能选择现有列表项之间，列表中第一项之前或最后一项之后的空间。不允许选择现有的列表项。
  • DropMode.ON_OR_INSERT是ON和INSERT模式。

的JTree类提供相同的放置模式集，并且JTable该类还有一些特定于添加行或列的方法。

要获取放置位置， TransferSupport类提供了getDropLocation返回下降发生的精确点的方法。但是对于列表组件，放置索引比像素位置更有用，因此JList提供了一个特殊的子类，称为JList.DropLocation 。此类提供了getIndex和isInsert方法，可以为您处理数学。
