如何使用密码字段

的JPasswordField类，是JTextField ，提供用于输入密码的专用文本字段。出于安全原因，密码字段不显示用户键入的字符。而是，该字段显示与键入的字符不同的字符，例如星号“ *”。作为另一项安全预防措施，密码字段将其值存储为字符数组而不是字符串。与普通文本字段类似，密码字段会在用户指示文本输入完成（例如通过按Enter按钮）时触发一个动作事件 。

passwordField = new JPasswordField(10);
passwordField.setActionCommand(OK);
passwordField.addActionListener(this);

该论点传递到JPasswordField构造函数指示字段的首选大小，在这种情况下，该大小至少为10列宽。默认情况下，密码字段为每个键入的字符显示一个点。如果要更改回显字符，请调用setEchoChar方法。然后，代码将操作侦听器添加到密码字段，该字段将检查用户键入的值。这是动作监听器的实现actionPerformed方法：

public void actionPerformed(ActionEvent e) {
    String cmd = e.getActionCommand();

    if (OK.equals(cmd)) { //Process the password.
        char[] input = passwordField.getPassword();
        if (isPasswordCorrect(input)) {
            JOptionPane.showMessageDialog(controllingFrame,
                "Success! You typed the right password.");
        } else {
            JOptionPane.showMessageDialog(controllingFrame,
                "Invalid password. Try again.",
                "Error Message",
                JOptionPane.ERROR_MESSAGE);
        }

        //Zero out the possible password, for security.
        Arrays.fill(input, '0');

        passwordField.selectAll();
        resetFocus();
    } else ...//handle the Help button...
}
安全说明：为了进一步提高安全性，当您完成由getPassword方法，应将其每个元素设置为零。前面的代码片段显示了如何执行此操作。

使用密码字段的程序通常会在完成要求密码的任何操作之前先验证密码。该程序调用私有方法， isPasswordCorrect ，用于比较getPassword方法将值存储到字符数组中。这是它的代码：

private static boolean isPasswordCorrect(char[] input) {
    boolean isCorrect = true;
    char[] correctPassword = { 'b', 'u', 'g', 'a', 'b', 'o', 'o' };

    if (input.length != correctPassword.length) {
        isCorrect = false;
    } else {
        isCorrect = Arrays.equals (input, correctPassword);
    }

    //Zero out the password.
    Arrays.fill(correctPassword,'0');

    return isCorrect;
}
