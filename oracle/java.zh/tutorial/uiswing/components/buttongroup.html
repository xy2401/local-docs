如何使用ButtonGroup组件

的ButtonGroup组件管理一组按钮的选中/未选中状态。对于小组来说ButtonGroup实例保证一次只能选择一个按钮。

最初，所有按钮都由ButtonGroup实例未选中。

如何使用ButtonGroup功能

您可以使用ButtonGroup与任何从其继承的对象集AbstractButton 。通常，按钮组包含以下实例JRadioButton, JRadioButtonMenuItem ， 要么JToggleButton 。放置一个实例是没有意义的JButton要么JMenuItem在按钮组中，因为JButton和JMenuItem不要实现选择/取消选择按钮的状态。

通常，您通常会按照以下步骤编写使用ButtonGroup零件。

  1. 子类JFrame
  2. 呼叫ContextPane与布局管理器一起
  3. 声明并配置一组单选按钮或切换按钮
  4. 实例化一个ButtonGroup宾语
  5. 致电add该按钮组对象上的方法，以便将每个按钮添加到组中。

有关详细信息和代码示例，请参见如何使用单选按钮 。它显示了如何使用ButtonGroup组件将一组RadioButtons组合到一个JPanel中。
