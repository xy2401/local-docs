B.3.1 Installation on MS-Windows

This section covers installation and usage of gawk on Intel architecture machines running any version of MS-Windows. In this section, the term “Windows32” refers to any of Microsoft Windows 95/98/ME/NT/2000/XP/Vista/7/8/10.

See also the README_d/README.pc file in the distribution.
