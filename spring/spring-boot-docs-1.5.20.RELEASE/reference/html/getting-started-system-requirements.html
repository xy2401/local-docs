9. System Requirements

By default, Spring Boot 1.5.20.RELEASE requires Java 7 and Spring Framework 4.3.23.RELEASE or above. You can use Spring Boot with Java 6 with some additional configuration. See Section 85.11, “How to use Java 6” for more details.

Explicit build support is provided for the following build tools:

Build ToolVersion

Maven

3.2+

Gradle

[2.9, 3.x]

Tip

Although you can use Spring Boot with Java 6 or 7, we generally recommend Java 8 if at all possible.
